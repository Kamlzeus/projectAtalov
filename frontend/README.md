# 🎨 Проект

Это веб-приложение на React + Flask, использующее обученную модель ResNet50 для автоматического определения художественного стиля картины по изображению.

## Используемые технологии:

- React (frontend)
- Flask + PyTorch (backend)
- Drag & Drop загрузка изображений

## 🧠 Классы, которые определяет модель:

- 🎨 Baroque
- 🎨 Cubism
- 🎨 Expressionism
- 🎨 Impressionism
- 🎨 Minimalism
- 🎨 Pop Art
- 🎨 Post-Impressionism
- 🎨 Realism
- 🎨 Rococo
- 🎨 Romanticism

## Запуск проекта:

### 1. Backend (Flask)

cd backend
pip install -r requirements.txt
python app.py

### 2. Frontend (React)

cd frontend
npm install
npm start
